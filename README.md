# Setup

### Variáveis de ambiente
As variáveis de ambiente podem ser definidas no arquivo `.env` e são carregadas automaticamente.

- `OWNER_ID`<br>
- `TOKEN`<br>
- `PREFIX_COMMAND`<br>
- `GUILD_LOG`<br> (server's ID)


```bash
cd kosaki
npm install
npm start
```
